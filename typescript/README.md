# TypeScript Examples

TypeScript versions of key patterns from this course. Run with `npx tsx <file>`.

## Setup

```bash
npm init -y
npm install openai anthropic
npm install -D typescript @types/node tsx
```

## Files

| File | Module | Description |
|------|--------|-------------|
| `chat.ts` | 00 | Basic LLM call with OpenAI/Anthropic |
| `structured-output.ts` | 01 | JSON schema extraction |
| `rag.ts` | 02 | Simple RAG pipeline |
| `agent.ts` | 07 | Tool-calling agent |
| `streaming.ts` | 05 | SSE streaming response |
